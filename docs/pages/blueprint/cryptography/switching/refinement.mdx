# The Refinement Polynomial [A virtual operation that "splits" a committed large-field polynomial]

In this page, we discuss _refinement_ and _packing_, two dual notions that sit at the center of ring-switching. Indeed, as we've said, we've thus far only [committed](/blueprint/cryptography/commitment) large-field polynomials. On the other hand, given any large-field polynomial, I can _freely decide to view it_ as a small-field polynomial, merely by refining the boundary points which divide distinct field elements. The point is to make this formal data-casting operation _algebraic_, so that we can treat it using the ring-switching reduction.
