# Binary Fields [Basics on the fields we use]

A _field_ is a set equipped with addition and multiplication operations, which satisfy the commutative, associative and distributive laws, and where everything nonzero has an inverse. A _finite field_ is moreover, well, finite.

The finite fields you're probably most used to look like $\mathbb{F}_p = \{0, \ldots , p - 1\}$, where $p$ is a large prime. Arithmetic in that set is carried out modulo $p$. Elements of the field can be represented, concretely, as $\lceil \log_2 p \rceil$-bit strings. Some of those strings—i.e., the ones representing integers in the high range $\{p, \ldots , \lceil \log_2 p \rceil - 1\}$—will be "off-limits".

## Binary Fields

Here, we work in characteristic 2. This means that we construct our fields differently.

The simplest binary field is $\mathbb{F}_2 = \{0, 1\}$, the field with two elements.

By taking algebraic extensions of this field, we can get further ones. We pick a bit-length. For us, we'll stick with $128$. If we pick an irreducible polynomial $f(X) \in \mathbb{F}_2[X]$ of degree $128$, then the quotient ring $\mathbb{F}_2[X] / (f(X))$ is a field. Up to ring isomorphism, this thing is unique. This gives us the field $\mathbb{F}_{2^{128}}$.

Each element $\alpha \in \mathbb{F}_{2^{128}}$ can be represented as a polynomial of degree less than $128$, say $\alpha = \sum_{i = 0}^{127} \alpha_i \cdot X^i$. Thus we can represent elements of $\mathbb{F}_{2^{128}}$ _concretely_ as $128$-bit strings (i.e., in coefficients with respect to the $\mathbb{F}_2$-basis $1, X, \ldots , X^{127}$).

## GHASH

Concretely, we use the _[GHASH](https://datatracker.ietf.org/doc/html/rfc8452#appendix-A)_ field. To get that field, we just need to take $f(X) \coloneqq X^{128} + X^7 + X^2 + X + 1$. That polynomial $f(X)$ is indeed irreducible in $\mathbb{F}_2[X]$.

On processors that feature the _[carryless multiplication](https://en.wikipedia.org/wiki/CLMUL_instruction_set) (CLMUL)_ instruction set extension, multiplication in the GHASH field can be carried out extremely efficiently.
