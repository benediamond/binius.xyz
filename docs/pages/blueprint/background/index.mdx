# Mathematical Background [Review of concepts necessary for our blueprint]

For each integer $\ell \geq 0$, the _$\ell$-dimensional cube_ $\mathcal{B}_\ell$ denotes the set of all bitstrings of length $\ell$ (i.e., the set $\{0, 1\}^\ell$). For each $\iota \geq 0$, the symbol $\mathcal{T}_\iota$ denotes a _binary tower field_ of height $\iota$. This thing is a _field_—i.e., a set containing things we can multiply and divide—whose elements each look like binary strings of length $2^\iota$.
