# Mathematizing the Problem [From AND constraints to multilinear polynomials]

In this page, we recast the problem of [AND constraint](/blueprint/constraints/ands) satisfaction into one about multilinears.

## The Constraint Matrices

We fix prover data $w$, of length $n_\text{words}$, and a set of AND constraints. As we've already explained, these amount to a list of $n_\text{and}$ triples $(L_a, L_b, L_c)$, indexed by $x \in \{0, \ldots , n_\text{and} - 1\}$. Each triple's three lists $L_a$, $L_b$ and $L_c$ are themselves made up of [shifted value indices](/blueprint/constraints/indices), triples of the form $(y, \text{op}, s) \in \{0, \ldots , n_\text{words} - 1\} \times \{\mathsf{sll}, \mathsf{sra}, \mathsf{sra}\} \times \{0, \ldots , 63\}$.

We first re-express this data in matrix form. We are going to write down a bunch of binary-valued matrices. For _each_ $\text{op} \in \{\mathsf{sll}, \mathsf{sra}, \mathsf{sra}\}$, we are going to write down an $n_\text{and} \times n_\text{words} \times 64$ "matrix" $A_{\text{and}, \text{op}}$—so, $A_{\text{and}, \text{op}}$ is three-dimensional—by declaring that, for each $x \in \{0, \ldots , n_\text{and} - 1\}$, $y \in \{0, \ldots , n_\text{words} - 1\}$, and $s \in \{0, \ldots , 63\}$,

$$
\begin{equation*}A_{\text{and}, \text{op}}[x, y, s] = 1\end{equation*}
$$

if and only if the $x$<sup>th</sup> constraint triple $(L_a, L_b, L_c)$'s first component $L_a$ contains the triple $(y, \text{op}, s)$. In other words: "Does the $x$<sup>th</sup> constraint's '$A$ part' contain $\text{op}(w[y], s)$ as an XOR-summand?" We create similar three-dimensional "matrices" $B_{\text{and}, \text{op}}$ and $C_{\text{and}, \text{op}}$.

Thus, we get $3 \times 3 = 9$ total three-dimensional matrices, namely $A_{\text{and}, \text{op}}$, $B_{\text{and}, \text{op}}$ and $C_{\text{and}, \text{op}}$ for _each_ $\text{op}$. Note that we are implicitly assuming there are no duplicates triples in any of the lists $(L_a, L_b, L_c)$ (for any $x$). Of course, having duplicates would be pointless, since these would get XOR'd out.

## Sum Expressions

We now define three arrays, $a$, $b$ and $a$, each of length $n_\text{and}$, and containing 64-bit words. For these, we use the [shifted witness functions](/blueprint/backend/shifts/polynomials) $\textsf{shift}_\text{op}(w)(y, s)$. For each $x \in \{0, \ldots , n_\text{and} - 1\}$, we define:

$$
\begin{align*}
a[x] &\coloneqq \sum_\text{op} \sum_{s = 0}^{63} \sum_{y = 0}^{n_\text{words} - 1} A_{\text{and}, \text{op}}[x, y, s] \cdot \textsf{shift}_\text{op}(w)(y, s), \\
b[x] &\coloneqq \sum_\text{op} \sum_{s = 0}^{63} \sum_{y = 0}^{n_\text{words} - 1} B_{\text{and}, \text{op}}[x, y, s] \cdot \textsf{shift}_\text{op}(w)(y, s), \\
c[x] &\coloneqq \sum_\text{op} \sum_{s = 0}^{63} \sum_{y = 0}^{n_\text{words} - 1} B_{\text{and}, \text{op}}[x, y, s] \cdot \textsf{shift}_\text{op}(w)(y, s).
\end{align*}
$$

It follows essentially by definition that $w$ satisfies the constraint system if and only if $a[x] \mathrel{\&} b[x] = c[x]$ holds for _each_ $x \in \{0, \ldots , n_\text{and} - 1\}$.

There is a straightforward efficient algorithm that computes $a$, $b$ and $c$, which exploits the sparsity of the constraint system. On input a constraint system and a length-$n_\text{words}$ vector of 64-bit prover data $w$, the prover runs:

- allocate arrays $a$, $b$ and $c$, each containing $n_\text{words}$ 64-bit words, initially all 0s.
- **for** each $x \in \{0, \ldots , n_\text{and} - 1\}$ **do**:
  - load the $x$<sup>th</sup> tuple $(L_a, L_b, L_c)$ from the constraint system.
  - **for** each shifted value index $(y, \text{op}, s) \in L_a$ **do**: update $a[x] \mathrel{\oplus}= \text{op}(w[y], s)$.
  - **for** each shifted value index $(y, \text{op}, s) \in L_b$ **do**: update $b[x] \mathrel{\oplus}= \text{op}(w[y], s)$.
  - **for** each shifted value index $(y, \text{op}, s) \in L_c$ **do**: update $c[x] \mathrel{\oplus}= \text{op}(w[y], s)$.
- output the arrays $a$, $b$ and $c$.

## Multilinearizing

Now, we turn everything into polynomials. At this point, we assume that $n_\text{and} = 2^{\ell_\text{and}}$ and $n_\text{words} = 2^{\ell_\text{words}}$ are both powers of 2. As usual, $\mathcal{B}_\ell \coloneqq \{0, 1\}^\ell$ means the $\ell$-dimensional cube.

By identifying the matrices $A_{\text{and}, \text{op}}$, $B_{\text{and}, \text{op}}$ and $C_{\text{and}, \text{op}}$ with their multilinear extensions, we can view them now as multilinear polynomials $\widetilde{A}_{\text{and}, \text{op}}(X, Y, S)$, $\widetilde{B}_{\text{and}, \text{op}}(X, Y, S)$ and $\widetilde{C}_{\text{and}, \text{op}}(X, Y, S)$, each with coefficients in $\mathbb{F}_2$, and of $\ell_\text{and} + \ell_\text{words} + 6$ variables. Similarly, we have already seen that the shifted witness functions $\textsf{shift}_\text{op}(w)(y, s)$ can be multilinearized. Multilinearizing everything, we obtain three new polynomials:

$$
\begin{align*}
\widetilde{a}(I, X) &\coloneqq \sum_\text{op} \sum_{s \in \mathcal{B}_6} \sum_{y \in \mathcal{B}_{\ell_\text{words}}} \widetilde{A}_{\text{and}, \text{op}}(X, y, s) \cdot \widetilde{\textsf{shift}}_\text{op}(w)(I, y, s), \\
\widetilde{b}(I, X) &\coloneqq \sum_\text{op} \sum_{s \in \mathcal{B}_6} \sum_{y \in \mathcal{B}_{\ell_\text{words}}} \widetilde{B}_{\text{and}, \text{op}}(X, y, s) \cdot \widetilde{\textsf{shift}}_\text{op}(w)(I, y, s), \\
\widetilde{c}(I, X) &\coloneqq \sum_\text{op} \sum_{s \in \mathcal{B}_6} \sum_{y \in \mathcal{B}_{\ell_\text{words}}} \widetilde{C}_{\text{and}, \text{op}}(X, y, s) \cdot \widetilde{\textsf{shift}}_\text{op}(w)(I, y, s).
\end{align*}
$$

It's again true essentially by definition that our constraint system is satisfied if and only if, for all inputs $i \in \mathcal{B}_6$ and $x \in \mathcal{B}_{\ell_\text{and}}$ in the cube, the identity

$$
\begin{equation*}\widetilde{a}(i, x) \cdot \widetilde{b}(i, x) - \widetilde{c}(i, x) \stackrel{?}= 0.\end{equation*}
$$

holds. Here, the $\cdot$ is _field multiplication_, which in this setting takes place between $\mathbb{F}_2$-elements necessarily. Note now that we are quantifying _both_ over word indices $x \in \mathcal{B}_{\ell_\text{and}}$ _and_ bit indices $i \in \mathcal{B}_6$.

We have officially gotten something that we can zerocheck.

## Using Shift Indicators

In order to prepare the way for what's coming, we do one further step. We [showed already](/blueprint/backend/shifts/polynomials) that the shifted witness polynomials $\widetilde{\textsf{shift}}_\text{op}(w)(I, y, S)$ can themselves be expressed using shift indicators. Unrolling that step now, we obtain the following  equivalent expressions for $\widetilde{a}(I, X)$, $\widetilde{b}(I, X)$ and $\widetilde{C}(I, X)$:

$$
\begin{align*}
\widetilde{a}(I, X) &= \sum_\text{op} \sum_{s \in \mathcal{B}_6} \sum_{y \in \mathcal{B}_{\ell_\text{words}}} \sum_{j \in \mathcal{B}_6} \widetilde{A}_{\text{and}, \text{op}}(X, y, s) \cdot \widetilde{\textsf{shift-ind}}_\text{op}(I, j, s) \cdot \widetilde{w}(j, y), \\
\widetilde{b}(I, X) &= \sum_\text{op} \sum_{s \in \mathcal{B}_6} \sum_{y \in \mathcal{B}_{\ell_\text{words}}} \sum_{j \in \mathcal{B}_6} \widetilde{B}_{\text{and}, \text{op}}(X, y, s) \cdot \widetilde{\textsf{shift-ind}}_\text{op}(I, j, s) \cdot \widetilde{w}(j, y), \\
\widetilde{c}(I, X) &= \sum_\text{op} \sum_{s \in \mathcal{B}_6} \sum_{y \in \mathcal{B}_{\ell_\text{words}}} \sum_{j \in \mathcal{B}_6} \widetilde{C}_{\text{and}, \text{op}}(X, y, s) \cdot \widetilde{\textsf{shift-ind}}_\text{op}(I, j, s) \cdot \widetilde{w}(j, y).
\end{align*}
$$

These are identically equal (as polynomials) to the above expressions we gave; they just unroll the shifted witness polynomials and rearrange the sums.
