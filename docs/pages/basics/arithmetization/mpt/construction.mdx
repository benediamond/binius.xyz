# Our MPT Construction [How we use M3 to arithmetize this task]

Here, we begin our description in earnest. For the sake of this page, we describe how we arithmetize the proof of just a _single_ key–value pair against the global state trie root. In our full-blown arithmetization, we need to prove _many_ inclusions at once. We manage to save work in that latter setting using a technique called [forking](/basics/arithmetization/mpt/forking); for now, we ignore this technicality.
