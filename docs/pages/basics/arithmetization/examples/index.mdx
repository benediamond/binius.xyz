# M3 Examples [Full-blown M3 instances used in Binius]

In this page, we discuss _actual examples_ of M3 instances, used in critical roles throughout Binius.

## Subsection Directory

This subsection has the following pages:

- [**Lasso lookup.**](lasso) The _Lasso_ lookup protocol [[STW23](https://eprint.iacr.org/2023/1216)], adapted for Binary fields in [[DP23, § 4.4](https://eprint.iacr.org/2023/1784.pdf)], can be viewed as an M3 instance in disguise. In fact, it's one of our most essential nontrivial applications of M3, and gets used extensively throughout Binius.
- [**RAM.**](ram) Here, we enrich the lookup problem, by allowing the prover to _write_ into a fixed RAM, as opposed to just reading from it.
- [**Merkle–Patricia inclusion.**](mpt) This is our "full-blown" M3 instance; it fully showcases the power and flexibility of M3 on a real-life problem of critical importance. Here, we want the prover to prove that a _public_ list of Ethereum account state values admit proper MPT inclusion proofs, via-à-vis a _public_ Ethereum state trie root. The _witnesses_ are the inclusion proofs / paths themselves, which need to be hidden (for efficiency reasons).
