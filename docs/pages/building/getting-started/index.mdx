# Getting Started

Build applications with Binius64 circuits. This guide takes you from fundamentals to production applications.

Follow this progression to build circuit-based applications:

1. **[Basics](basics)** - Core concepts: verification, circuit hierarchy, gates, constraints, and examples. Learn how zero-knowledge circuits work and build understanding through working examples.

2. **[Installation](installation)** - Set up Binius64 and run examples. Get your development environment ready and explore the available circuit implementations.

3. **[First Circuit](first-circuit)** - Build your first circuit with basic gates. Create a working circuit that demonstrates constraint verification and witness generation.

4. **[Proving Workflow](proving)** - Generate and verify proofs programmatically. Transform constraint verification into cryptographic proofs that convince third parties.

5. **[CLI Framework](cli)** - Turn circuits into command-line applications with proof generation. Package your circuits as reusable command-line tools for production use.
