# First Circuit

Start by creating a new project to apply the concepts from [Basics](basics):

```bash
cargo new my-circuit
cd my-circuit

# Add dependencies to Cargo.toml
cat >> Cargo.toml << 'EOF'
binius-frontend = { git = "https://github.com/IrreducibleOSS/binius64" }
binius-core = { git = "https://github.com/IrreducibleOSS/binius64" }
EOF
```

## Build the Circuit

Create `src/main.rs` implementing the masked AND example from basics:

```rust
use binius_frontend::compiler::CircuitBuilder;
use binius_core::{word::Word, verify::verify_constraints};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Phase 1: Circuit Building
    let builder = CircuitBuilder::new();
    
    let mask = builder.add_constant_64(0xFF00);
    let private = builder.add_witness();  // Private value
    let result = builder.band(private, mask);
    
    // Public output for verification
    let output = builder.add_inout();
    builder.assert_eq("masked_result", result, output);
    
    let circuit = builder.build();
    
    // Phase 2: Witness Generation
    let mut witness = circuit.new_witness_filler();
    witness[private] = Word(0x1234);  // Secret: 0x1234
    witness[output] = Word(0x1200);   // Public: result of 0x1234 & 0xFF00
    
    circuit.populate_wire_witness(&mut witness)?;
    
    // Phase 3: Constraint Verification
    let cs = circuit.constraint_system();
    verify_constraints(cs, &witness.into_value_vec())?;
    
    println!("✓ Proof verified: Someone knows a value that masks to 0x1200");
    println!("  Circuit used {} AND constraints", cs.n_and_constraints());
    
    Ok(())
}
```

## Run and Verify

```bash
RUSTFLAGS="-C target-cpu=native" cargo run --release
```

Expected output:
```
✓ Proof verified: Someone knows a value that masks to 0x1200
  Circuit used 2 AND constraints
```

## Experiment

Modify the circuit to explore different patterns:

```rust
// Try different operations from basics examples
let xor_result = builder.bxor(a, b);                       // Field addition
let (sum, _) = builder.iadd_cin_cout(x, y, zero);         // Integer addition with carry
let cmp = builder.icmp_ult(p, q);                         // Comparison (returns boolean)

// XOR packing optimization
let packed_xor = builder.bxor_multi(&[a, b, c, d]);       // Single constraint for all XORs

// Multiple constraints
builder.assert_eq("check1", computed1, expected1);
builder.assert_eq("check2", computed2, expected2);

// Public inputs with private computation
let public_input = builder.add_inout();
let secret = builder.add_witness();
let (computed, _) = builder.iadd_cin_cout(public_input, secret, zero);
builder.assert_eq("verify", computed, public_output);

// Non-deterministic hints (from Example 3)
let (quotient, remainder) = builder.biguint_divide_hint(&dividend, &divisor);
```
