# First Circuits

Start by creating a new project to apply the concepts from [Basics](basics):

```bash
cargo new my-circuit
cd my-circuit

# Add dependencies to Cargo.toml
cat >> Cargo.toml << 'EOF'
binius-frontend = { git = "https://github.com/IrreducibleOSS/binius64" }
binius-core = { git = "https://github.com/IrreducibleOSS/binius64" }
EOF
```

## Example 1: Masked AND Circuit

Create `src/main.rs` implementing the masked AND example from basics:

```rust
use binius_frontend::compiler::CircuitBuilder;
use binius_core::{word::Word, verify::verify_constraints};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Phase 1: Circuit Building
    let builder = CircuitBuilder::new();
    
    let mask = builder.add_constant_64(0xFF00);
    let private = builder.add_witness();
    let result = builder.band(private, mask);
    
    // Public output for verification
    let output = builder.add_inout();
    builder.assert_eq("masked_result", result, output);
    
    let circuit = builder.build();
    
    // Phase 2: Witness Generation
    let mut w = circuit.new_witness_filler();
    w[private] = Word(0x1234);
    w[output] = Word(0x1200);
    
    circuit.populate_wire_witness(&mut w)?;
    
    // Phase 3: Constraint Verification
    let cs = circuit.constraint_system();
    verify_constraints(cs, &w.into_value_vec())?;
    
    println!("✓ Proof verified: Someone knows a value that masks to 0x1200");
    println!("  Circuit used {} AND constraints", cs.n_and_constraints());
    
    Ok(())
}
```

Run and verify:

```bash
RUSTFLAGS="-C target-cpu=native" cargo run --release
```

Expected output:
```
✓ Proof verified: Someone knows a value that masks to 0x1200
  Circuit used 2 AND constraints
```

## Example 2: SHA256 Preimage Circuit

For a more practical example, implement a SHA256 preimage proof using the framework's built-in SHA256 circuit:

```rust
use binius_frontend::{compiler::CircuitBuilder, circuits::sha256::Sha256};
use binius_core::{word::Word, verify::verify_constraints};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Phase 1: Circuit Building
    let builder = CircuitBuilder::new();
    
    // 64-byte message (8 words)
    let message: Vec<_> = (0..8)
        .map(|_| builder.add_witness())
        .collect();
    
    // Private message length
    let message_len = builder.add_witness();
    
    // SHA256 digest (4 words = 256 bits)
    let expected_digest: [_; 4] = core::array::from_fn(|_| builder.add_inout());
    
    // Create SHA256 circuit
    let sha256 = Sha256::new(
        &builder,
        message_len,
        expected_digest,
        message.clone()
    );
    
    let circuit = builder.build();
    
    // Phase 2: Witness Generation
    let mut w = circuit.new_witness_filler();
    
    // Message length
    w[message_len] = Word(64);
    
    // Message with nonce
    let mut message_bytes = vec![0u8; 64];
    let test_data = b"test message data";
    message_bytes[..test_data.len()].copy_from_slice(test_data);
    // Add nonce to remaining bytes
    let nonce = 0x123456789ABCDEF0u64.to_le_bytes();
    let nonce_start = test_data.len();
    message_bytes[nonce_start..nonce_start + 8].copy_from_slice(&nonce);
    
    // Populate message
    sha256.populate_message(&mut w, &message_bytes);
    
    // Expected digest
    use sha2::{Digest, Sha256 as StdSha256};
    let hash = StdSha256::digest(&message_bytes);
    let mut digest_bytes = [0u8; 32];
    digest_bytes.copy_from_slice(&hash);
    
    sha256.populate_digest(&mut w, digest_bytes);
    
    circuit.populate_wire_witness(&mut w)?;
    
    // Phase 3: Constraint Verification
    let cs = circuit.constraint_system();
    verify_constraints(cs, &w.into_value_vec())?;
    
    println!("✓ SHA256 preimage verified without revealing the message or its length");
    println!("  Circuit used {} AND constraints", cs.n_and_constraints());
    
    Ok(())
}
```

Add the SHA256 dependency to `Cargo.toml`:
```toml
sha2 = "0.10"
```

Run and verify:
```bash
RUSTFLAGS="-C target-cpu=native" cargo run --release
```

Expected output:
```
✓ SHA256 preimage verified without revealing the message or its length
  Circuit used 4180 AND constraints
```

## Experiment

Modify the circuit to explore different patterns:

```rust
// Try different operations from basics examples
let xor_result = builder.bxor(a, b);
let (sum, _) = builder.iadd_cin_cout(x, y, zero);
let cmp = builder.icmp_ult(p, q);

// XOR packing optimization
let packed_xor = builder.bxor_multi(&[a, b, c, d]);

// Multiple constraints
builder.assert_eq("check1", computed1, expected1);
builder.assert_eq("check2", computed2, expected2);

// Public inputs with private computation
let public_input = builder.add_inout();
let secret = builder.add_witness();
let (computed, _) = builder.iadd_cin_cout(public_input, secret, zero);
builder.assert_eq("verify", computed, public_output);

// Non-deterministic hints (from Example 3)
let (quotient, remainder) = builder.biguint_divide_hint(&dividend, &divisor);
```
