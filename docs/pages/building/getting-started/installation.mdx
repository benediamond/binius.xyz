# Installation

Setting up Binius64 requires Rust 1.70+ with Cargo and Git for repository access.

## Production Examples

Clone the repository to run production circuit implementations:

```bash
git clone https://github.com/IrreducibleOSS/binius64
cd binius64

# List all examples
ls crates/examples/examples/

# Run SHA256 circuit (most comprehensive)
RUSTFLAGS="-C target-cpu=native" cargo run --release --example sha256

# Benchmark performance
RUSTFLAGS="-C target-cpu=native" cargo bench --bench sha256
```

## Tutorial Examples

Tutorial examples are located in `crates/examples/examples/tutorials/`:

```bash
# Run all tutorial examples
for f in crates/examples/examples/tutorials/*.rs; do n=$(basename $f .rs); echo "=== $n ===" && RUSTFLAGS="-C target-cpu=native" cargo run --release -p binius-examples --example $n || break; done

# Or run a specific tutorial example
RUSTFLAGS="-C target-cpu=native" cargo run --release -p binius-examples --example getting_started_first_circuits
```
