# Installation

Setting up Binius64 requires Rust 1.70+ with Cargo and Git for repository access.

## Production Examples

Clone the repository to run production circuit implementations:

```bash
git clone https://github.com/IrreducibleOSS/binius64
cd binius64

# List all examples
ls crates/examples/examples/

# Run SHA256 circuit (most comprehensive)
RUSTFLAGS="-C target-cpu=native" cargo run --release --example sha256

# Benchmark performance
RUSTFLAGS="-C target-cpu=native" cargo bench --bench sha256
```
Other examples you can try include `keccak`, `ethsign` and `blake2b`.
