# Circuits

1. **[Gadgets](gadgets)** - Reusable functions for common operations like byte manipulation and bit counting. Compose multiple gates into higher-level algorithms for complex tasks.

2. **[Wires](wires)** - Wire types and witness array organization for data flow between gates. Understand how data flows through circuits and compiler optimizations for wire management.

3. **[Control Flow](control-flow)** - Conditional execution and iteration patterns without runtime branching. Implement if-statements, loops, and array access using arithmetic masking and multiplexers.

4. **[Data Structures](data-structures)** - Dealing with data structures in circuits using static allocation. Handle arrays, concatenation, and variable-length data within circuit constraints.

5. **[Composition](composition)** - Circuit building patterns and organization for larger systems. Organize complex circuits through consistent function patterns, wire ownership, and hierarchical design.

6. **[Performance](performance)** - Analysis tools and optimization through compiler passes. Understand cost models, constraint counting, and circuit snapshots for performance analysis.
