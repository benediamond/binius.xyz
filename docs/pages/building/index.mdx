# Building with Binius64 [Tutorial on building ZK applications]

This guide covers core concepts through building applications in Binius64.

**[Getting Started](getting-started)** - Covers the fundamentals of zero-knowledge circuits including verification concepts, building your first circuit, generating proofs, and creating command-line applications. Shows how constraint systems work through practical examples.

**[Circuits](circuits)** - Explores practical patterns for building complex circuits including reusable gadgets, control flow without branches, circuit composition techniques, and performance analysis tools. These enable building production-ready cryptographic functions.

Production applications rely on complex cryptographic circuits for real-world use cases. The framework includes production implementations of:
- **Hash functions**: SHA256, SHA512, Blake2, Keccak, RIPEMD
- **Signatures**: ECDSA, secp256k1, RSA
- **Encoding**: Base64, JWT claims parsing
- **Arithmetic**: BigNum operations, floating point

For cryptographic and mathematical details, see the [Blueprint](/blueprint) section.
