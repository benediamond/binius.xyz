# Building [Guidance on the building Binius constraint systems]

Binius is a proof system framework that can be found and downloaded [here](https://gitlab.com/IrreducibleOSS/binius). To build on Binius you must build a constraint system. The following pages of documentation are intended as guidance for building Binius constraint systems, and connecting them with Binius [`prove`](https://docs.binius.xyz/binius_core/constraint_system/fn.prove.html) and [`verify`](https://docs.binius.xyz/binius_core/constraint_system/fn.verify.html) functions for a complete proof system.

- **[Constraint System](/building/constraints):** We outline the creation of a constraint system, and its interaction with Binius `prove` and `verify` functions.
- **[Bitwise Constraint](/building/bitwise):** We give a simple example of a constraint system for verifying bit operations.
- **[The Basic Pattern](/building/pattern):** The basic pattern utilized repeated in building Binius constraint systems is three fold: declare columns, populate them with values, and constrain those columns. We explain each of these steps in detail.
- **[Making Transparents](/building/transparents):** When the verifier (as well as the prover) holds a column, the column is called "transparent." Due their importance, for example in lookup constructions, we present a way to create them out of arbitrary values.
- **[Collatz in Practice](/building/collatz):** Channels and flushing are core to our arithmetization, and we illustrate their usage in a full proof system to complement the theoretical discussion in [Proving Collatz Orbits](/basics/arithmetization/matching/collatz).
- **[Making Lookups](/building/lookups):** For a second full proof system utilizing channels in a different way, we construct a gadget for lookups.

Beyond these simple guiding examples, many more examples of Binius constraint systems can be found in [Binius circuits](https://gitlab.com/IrreducibleOSS/binius/-/tree/main/crates/circuits/src) and for examples involving full prove and verify functionality see [Binius examples](https://gitlab.com/IrreducibleOSS/binius/-/tree/main/examples).
