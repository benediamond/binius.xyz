# Building with Binius64 [Tutorial on building ZK applications]

Tutorial on the building of ZK applications using Binius64.

## Overview

This section provides a comprehensive guide to building zero-knowledge circuits using the Binius64 framework. Binius64 is a simplified SNARK protocol that leverages binary field arithmetic to efficiently prove computations on 64-bit words.

### What You'll Learn

- **Circuit Foundations**. Why ZK systems require circuits and how they differ from traditional digital circuits.
- **Constraint Systems**. How to express computations using AND and MUL constraints.
- **Shifted Value Indices**. How Binius64 enables efficient bit manipulation.
- **Circuit Building**. Practical examples using the CircuitBuilder API.
- **Control Flow**. Patterns for handling conditionals and loops in circuits.
- **Proving and Verification**. The complete flow from circuit to proof.

### Binius64 Characteristics

1. **Word-Level Operations**. Works directly with 64-bit words instead of individual bits.
2. **CPU Alignment**. Maps to modern processor instructions.
3. **Two Constraint Types**. AND and MUL constraints.
4. **Zero-Cost Operations**. Shifts and XORs through shifted value indices.

### Prerequisites

- Basic understanding of zero-knowledge proofs concepts
- Familiarity with Rust programming
- Understanding of Boolean logic and basic arithmetic
